# TFM Baxter – Códigos

Scripts usados en el TFM para identificación dinámica del robot Baxter.

## Estructura
- `VIRTUAL/`: generación de datasets y entrenamiento/validación en simulación.
- `REAL/`: preprocesado, entrenamiento/validación y comparativas en datos reales.

## Nota
Este repositorio contiene únicamente el código (scripts). No incluye datasets ni outputs pesados.
